# 股票面板开发文档

## 1. 功能概述
股票面板是面向股票投资者的实时行情工具，提供大盘指数监控、自选股管理、个股详情查看、行情排行等核心功能，支持实时数据刷新和交互操作。

### 2.2 核心功能模块
| 模块 | 功能说明 | 交互逻辑 |
|------|----------|----------|
| 顶部导航 | 提供股票搜索、页面跳转入口 | 搜索框输入股票代码/名称实时联想，点击导航项切换对应页面 |
| 大盘指数 | 展示三大指数实时行情 | 点击指数跳转到对应指数详情页，数据每3秒刷新一次 |
| 自选股列表 | 展示用户关注的股票实时数据 | 点击股票切换右侧个股详情，左滑支持删除自选，下拉刷新数据 |
| 个股详情 | 展示选中股票的K线图和交易数据 | 支持切换K线周期，点击K线显示对应时间点的交易数据 |
| 买卖盘口 | 展示五档买卖挂单数据 | 数据每2秒刷新一次，高亮显示大额挂单 |
| 行情排行 | 展示各类股票排行榜单 | 点击榜单标签切换排行类型，点击股票跳转到个股详情 |
| 底部导航 | 页面快速切换 | 点击切换到对应功能页面 |

## 3. 接口规范

### 3.1 通用规则
- 协议：HTTPS
- 域名：api.stock.example.com
- 数据格式：JSON
- 签名方式：请求头携带`Authorization: Bearer {token}`
- 响应通用结构：
  ```json
  {
    "code": 0, // 0:成功 非0:错误码
    "msg": "success", // 错误信息
    "data": {} // 返回数据
  }
  ```

### 3.2 接口列表

#### 3.2.1 获取大盘指数
- **接口地址**：GET /api/v1/market/index
- **功能说明**：获取上证指数、深证成指、创业板指实时行情
- **请求参数**：无
- **返回示例**：
  ```json
  {
    "code": 0,
    "msg": "success",
    "data": [
      {
        "code": "sh000001",
        "name": "上证指数",
        "price": 3215.56,
        "change": 26.12,
        "changePercent": 0.82,
        "volume": 2345.67,
        "amount": 3456.78
      },
      {
        "code": "sz399001",
        "name": "深证成指",
        "price": 11234.67,
        "change": 116.89,
        "changePercent": 1.05,
        "volume": 3456.78,
        "amount": 4567.89
      }
    ]
  }
  ```

#### 3.2.2 获取自选股列表
- **接口地址**：GET /api/v1/user/self-stocks
- **功能说明**：获取用户自选股实时行情列表
- **请求参数**：无
- **返回示例**：
  ```json
  {
    "code": 0,
    "msg": "success",
    "data": [
      {
        "code": "600519",
        "name": "贵州茅台",
        "price": 1780.00,
        "change": 40.68,
        "changePercent": 2.34,
        "volume": 23000,
        "amount": 40.94,
        "high": 1785.00,
        "low": 1735.00,
        "open": 1742.00
      }
    ]
  }
  ```

#### 3.2.3 股票搜索
- **接口地址**：GET /api/v1/stock/search
- **功能说明**：根据关键词搜索股票
- **请求参数**：
  | 参数名 | 类型 | 必传 | 说明 |
  |--------|------|------|------|
  | keyword | string | 是 | 股票代码/名称拼音/全称 |
- **返回示例**：
  ```json
  {
    "code": 0,
    "msg": "success",
    "data": [
      {
        "code": "002594",
        "name": "比亚迪",
        "pinyin": "byd",
        "market": "sz"
      }
    ]
  }
  ```

- **接口地址**：GET /api/v1/stock/quote
- **功能说明**：获取单个股票实时行情数据

      "price": 238.90,
      "change": -2.10,

      "amount": 29.86,

#### 3.2.5 获取个股K线数据
- **接口地址**：GET /api/v1/stock/kline
- **功能说明**：获取股票K线历史数据
- **请求参数**：
  | 参数名 | 类型 | 必传 | 说明 |
  |--------|------|------|------|
  | code | string | 是 | 股票代码 |
  | period | string | 是 | K线周期：day/week/month/min1/min5/min15/min30/min60 |
  | count | int | 否 | 返回数据条数，默认100 |
- **返回示例**：
  ```json
  {
    "code": 0,
    "msg": "success",
    "data": [
      {
        "time": "2026-04-29",
        "open": 241.00,
        "high": 243.50,
        "low": 236.80,
        "close": 238.90,
        "volume": 125000
      }
    ]
  }
  ```

#### 3.2.7 获取行情排行榜
- **接口地址**：GET /api/v1/market/rank
- **功能说明**：获取各类股票排行榜数据
- **请求参数**：
  | 参数名 | 类型 | 必传 | 说明 |
  |--------|------|------|------|
  | type | string | 是 | 排行类型：increase/decrease/amount/turnover |
  | count | int | 否 | 返回条数，默认20 |
- **返回示例**：
  ```json
  {
    "code": 0,
    "msg": "success",
    "data": [
      {
        "rank": 1,
        "code": "300XXX",
        "name": "某某科技",
        "price": 25.60,
        "changePercent": 10.02,
        "volume": 123456
      }
    ]
  }
  ```

#### 3.2.8 操作自选股
- **接口地址**：POST /api/v1/user/self-stock
- **功能说明**：添加/删除自选股
- **请求参数**：
  | 参数名 | 类型 | 必传 | 说明 |
  |--------|------|------|------|
  | code | string | 是 | 股票代码 |
  | action | string | 是 | 操作类型：add/delete |
- **返回示例**：
  ```json
  {
    "code": 0,
    "msg": "success",
    "data": {}
  }
  ```

## 4. 技术要求
- 实时数据刷新频率：大盘/自选股/盘口数据每3秒刷新一次
- 数据缓存策略：K线数据缓存5分钟，排行数据缓存1分钟
- 错误处理：网络异常时显示缓存数据，重试机制最多3次
- 性能要求：页面首屏加载时间≤2s，滑动流畅度≥60fps
